# iCloud Shared Album Downloader

GUI-приложение для скачивания фото и видео из публичного общего альбома iCloud
(ссылки вида `https://www.icloud.com/sharedalbum/#TOKEN`).

## Возможности
- Качает оригиналы фото и видео в несколько потоков
- Определяет тип файла по содержимому (jpg / png / heic / mov / mp4) — видео не превращаются в «битые .jpg»
- Пропускает уже скачанное; можно прерывать и продолжать
- В конце проверяет целостность (наличие, размер, реальный тип) и чинит расширения
- Папка по умолчанию подставляется по названию самого альбома

## Готовый exe
Скачать собранный `.exe` — во вкладке [Releases](../../releases).

## Запуск из исходника
Нужен Python 3.8+ (tkinter входит в стандартную поставку Windows). Зависимостей нет.
```
python icloud_downloader.py
```

## Сборка exe вручную
```
pip install pyinstaller
pyinstaller --onefile --windowed --name iCloud-Downloader icloud_downloader.py
```
Результат — `dist/iCloud-Downloader.exe`.

## Использование
1. Открыть приложение
2. Вставить ссылку на общий альбом
3. Папку оставить пустой (возьмётся имя альбома) или указать свою
4. Нажать «Скачать»
